Eend Detectie Software™

De software voor eendherkenning is ontwikkeld door een team van studenten van Avans Hogeschool. De software is ontwikkeld om eenden te herkennen en te tellen. De software is ontwikkeld met de programmeertaal Python. De herkenningssoftware maakt gebruik van een model dat is getraind met een dataset van 1000 foto's van eenden. De dataset is gemaakt door de studenten zelf. De dataset is gemaakt met behulp van een Raspberry Pi en een camera. De software is gebaseerd op de techniek van object detection. Object detection is een techniek om objecten te herkennen in een afbeelding. De software is getest met behulp van een Raspberry Pi en een camera. De software is getest met een dataset van 200 foto's van eenden.
